# ĐỀ : XÂY DỰNG WEBSITE THƯƠNG MẠI ĐIỆN TỬ SỬ DỤNG PORTGRESS + ODOO
### ***1. Lý thuyết và Công nghệ sử dụng (Tech Stack)***
### Phần này giúp khẳng định nền tảng kỹ thuật vững chắc của dự án:

- Odoo Framework (phiên bản Community):

- Là một nền tảng mã nguồn mở quản trị doanh nghiệp (ERP) toàn diện.

- Sử dụng ngôn ngữ lập trình Python (phù hợp với kỹ năng bạn đang học) và kiến trúc Model-View-Controller (MVC).

- Cung cấp module Website và eCommerce tích hợp sẵn, cho phép quản lý từ giao diện bán hàng đến kho vận và kế toán trên cùng một hệ thống.

### PostgreSQL (Database Layer):

- Hệ quản trị cơ sở dữ liệu đối tượng - quan hệ (ORDBMS) mạnh mẽ, là yêu cầu bắt buộc để vận hành Odoo.

- Hỗ trợ các kiểu dữ liệu phức tạp và có tính toàn vẹn dữ liệu cực cao, phù hợp cho các giao dịch tài chính thương mại.

### Docker & Docker Compose:

- Tiếp tục sử dụng Docker để đóng gói ứng dụng, đảm bảo tính nhất quán giữa môi trường phát triển và triển khai.

- Sử dụng Docker Volume để lưu trữ dữ liệu bền vững (Persistent Data) cho cả Postgres và Odoo.

### ***2. Cấu trúc dự án (Project Structure)***
```
~/projects/odoo-ecommerce/
├── docker-compose.yml       # File cấu hình định nghĩa dịch vụ Odoo và Postgres.
├── .env                     # Lưu trữ mật khẩu và các biến môi trường bảo mật.
├── config/                  # Thư mục chứa file cấu hình odoo.conf (nếu cần tùy chỉnh sâu).
├── addons/                  # Thư mục quan trọng nhất để chứa các Module tùy chỉnh.
└── data/                    # Thư mục lưu trữ dữ liệu vật lý (được mapping qua volumes).
    ├── postgres/            # Dữ liệu của cơ sở dữ liệu PostgreSQL.
    └── odoo_web/            # Các tệp tin static, ảnh sản phẩm, filestore của Odoo.
```

### ***3. Quy trình triển khai (Workflow)***
Bạn có thể liệt kê các bước này vào phần "Implementation":

- Thiết lập môi trường: Khởi tạo cấu trúc thư mục trên Ubuntu bằng các lệnh Linux cơ bản (mkdir, cd).

- Định nghĩa hạ tầng: Xây dựng file docker-compose.yml thiết lập kết nối giữa Odoo container và Postgres container thông qua mạng nội bộ.

- Khởi tạo Database: Truy cập giao diện quản trị Odoo để khởi tạo Database gốc, thiết lập Master Password để bảo mật hệ thống.

- Triển khai Module: Kích hoạt module Website và thực hiện quy trình thiết kế giao diện (Drag-and-drop) chuyên nghiệp của Odoo.
